# PRIVACY.md

## Princípios
- Coleta mínima de dados.
- Finalidade educacional explícita.
- Retenção limitada e revisão periódica.

## Dados coletados
- Conta: email, nome de exibição, provedor OAuth.
- Progresso: campanhas, sessões, achievements, event logs.
- Telemetria técnica mínima para segurança e qualidade.

## Dados que NÃO coletamos
- Conteúdo de dispositivos do usuário.
- Tráfego de rede real.
- Credenciais além do fluxo OAuth delegado.

## Retenção
- Logs de segurança: janela mínima necessária para auditoria.
- Dados de progresso: enquanto conta ativa ou até solicitação de exclusão.
- Exclusão/anônimização conforme política operacional do projeto.

## Compliance
- Referência conceitual a GDPR (data minimization, purpose limitation).
